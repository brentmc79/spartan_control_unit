## 1. Overall Style & Visual Language
This design represents a graphical user interface, displayed on a small 320x170px LCD, used to control integrated electronics within a scifi armor costume inspired by the Halo video game franchise. GUI interaction is via two physical buttons; one for traversing elements, and the other for selecting the active element.

Core characteristics:
- Dark background
- Computer terminal blue/gree color palette with minimal accent colors
- Futuristic scifi borders, with occasional greebles and asymetric chamfered corners

Target perception:
- Spartan
- Simple to navigate
- Dense, but still clear despite small display

Output:
- One mockup per menu
- No surrounding graphics, only what is on the display
- Ensure the 3:1 main/sidebar ratio is respected

---

## 2. Layout Structure

### Global Layout
- Two columns:
  1. Main
    - contains menu and interaction items
    - 240px wide
  2. Sidebar
    - displays system status
    - 80px wide

---

## 3. Menu Layout
- Vertical list of traversable menu items
- Clear indicator of "active" menu item
- All submenus should have an item to back out, either text or an arrow
- Four top-level menu items:
  1. Visor
  2. Thermals
  3. HUD
  4. Settings

---

## 4. Sidebar
- Width: 240px
- Contains icons indicating system status

Icons:
- Fan blades or wind, some way to indicate stationary vs moving air
- LED light, indicating color and on or off status

---
